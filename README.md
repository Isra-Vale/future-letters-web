# Future Letters Web

A simple web app where you can write messages to your future self and choose when they get unlocked.


## What this is

This project is basically a digital time capsule.  
You write a message today, pick a date, and come back to it later — days, months, or even years after.

It’s meant to capture thoughts, goals, or emotions in the moment and let you revisit them with a different mindset.


##  What it does

- Write letters to your future self  
- Choose when each message unlocks  
- Add a mood to each message (happy, calm, stressed, etc.)  
- See a timeline of all your messages  
- Read unlocked messages when the time comes  


## Built with

- HTML  
- CSS  
- JavaScript  


## What I focused on

While building this, I tried to focus more on the experience than just the code — making it feel personal and smooth to use.

I also worked on:
- structuring a multi-section UI (writing, dashboard, timeline)  
- handling time-based logic for unlocking messages  
- keeping the design clean and simple
